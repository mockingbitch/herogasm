# Game Change Log

> *"Every update is not just a patch — it is a shift in the world."*

# Versioning System

```text id="v0n3lp"
MAJOR.MINOR.PATCH
```

* **MAJOR**: thay đổi hệ thống (combat, economy, core mechanics)
* **MINOR**: thêm content (Hero, Dungeon, Boss, Rune)
* **PATCH**: balance, fix bug, QoL

# Design Philosophy in Updates

Every update follows:

```text id="z1x8qk"
Add Complexity → Observe Meta → Adjust Systems → Expand Content
```

Not:

```text id="badflow"
Add Power → Nerf Later → Break Builds → Repeat
```
