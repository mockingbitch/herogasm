# Game Balance Rules

## Philosophy

Balance should create long-term progression.

Players should always have:

- A short-term goal
- A medium-term goal
- A long-term goal

Every session should feel rewarding.

Every upgrade should feel meaningful.

# AI Instructions

When generating gameplay systems:

- Prefer meaningful choices over larger numbers.
- Avoid power creep.
- Keep all classes viable.
- Design resource sinks.
- Reward both active and idle play.
- Create long-term progression.
- Balance using percentages, not arbitrary values.
- Explain why a balance decision is made.
